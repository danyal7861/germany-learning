Accessibility

 

Please read these accessibility policies carefully before using HealthAdvice Group website operated by HealthAdvice Group.

 

Accesbility Statement

We're committed to making our websites accessible so it can be used by as many people as possible. This statement applies to content published within the HealthAdvice Group website, including all pages.

We aim to make our websites accessible to the widest audiences. For example, you should be able to:

zoom in up to 300% without the text spilling off the screen, navigate most of our websites using a keyboard, listen to most of our websites using a screen reader, Take advantage of all the accesbility features built into your browser as well as a dedicated colour invert button on all pages.

To make HealthAdvice Group accessible, in future versions we will:

integrate accessibility into our procedures, Include a dark mode for reduced eye strain, Include a built in text-to-speech engine.
